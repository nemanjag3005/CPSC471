CPSC471 Final Project - Gym management system
Contributers: Nemanja Grujic, River Sanoy, Abdullah Zubair

To run the application, node needs to be installed on your machine as well as postgresql.

To run the frontend (website), change the directory to "client" and type "npm install" inside the directory in the command line to install all
the necessary dependencies. After that, type in "npm start" to run the website at localhost3000.

To run the backend, change the directory to "server" and type "npm install" inside the directory in the command line to install all
the necessary dependencies. After that, type in "nodemon index" to start the server at localhost5000.
